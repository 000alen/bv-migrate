# Importing to Circle

The import step takes the finalized course structure and creates it in Circle. It streams progress over SSE so you can watch it build.

## What gets created

For each course:
- One Circle **course** (a `course` type space in your space group)
- One Circle **section** per top-level section in the extracted structure
- One Circle **lesson** per lesson, with HTML body content

The creation order is: course first, then sections in order, then lessons within each section in order. Circle's API requires the parent to exist before the child, so this is sequential.

## Lesson HTML

Each lesson's `body_html` is generated by `lib/html-builder.ts`. It converts the content blocks array to HTML. Each block type has its own rendering:

- `text` — raw HTML passthrough (Claude produces valid HTML in the `html` field)
- `heading` — `<h1>`–`<h4>` depending on `level`
- `image` — `<action-text-attachment>` with `sgid` attribute (Circle's Trix-compatible format)
- `quiz` — styled `<div>` with question, lettered options, and highlighted correct answer
- `flashcard` — front/back with a visual separator
- `accordion` — disclosure-style with title and body
- `timeline` — ordered list with date and description pairs
- `quote` — `<blockquote>` with optional attribution

The HTML is self-contained. Circle renders it inside the lesson editor's view.

## Progress streaming

The import route streams SSE events:

```
event: progress
data: {"message": "Creating course...", "step": 1, "total": 47}

event: progress
data: {"message": "Creating section: What is Product Management?", "step": 3, "total": 47}

event: progress
data: {"message": "Creating lesson: The PM Role", "step": 4, "total": 47}

event: complete
data: {"courseId": 12345, "courseUrl": "https://app.circle.so/..."}
```

The browser shows a progress bar based on `step` / `total`.

## Partial failures

If the import fails partway through — network error, Circle 5xx, rate limit that exhausted retries — the route returns a `partial` event with the resources created so far:

```json
{
  "created": {
    "courseId": 12345,
    "sections": [{"id": 67890, "title": "What is PM?"}],
    "lessons": [{"id": 11111, "title": "The PM Role", "sectionId": 67890}]
  },
  "error": "Failed to create lesson 'Strategy Frameworks': 503 Service Unavailable"
}
```

You can use this to manually complete the import or clean up the partial course in Circle. The tool doesn't retry a full import from scratch — it would create duplicates.

## Rate limits

Circle's API enforces rate limits. The tool paces requests with a small delay between lesson creates to avoid hitting them. See [Circle API](../technical/circle-api) for specifics.
